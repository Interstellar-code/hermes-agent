# SwitchUI Skill

This skill provides guidance on working with the SwitchUI browser frontend for Hermes — including how to start it, configure environment variables, understand its features (chat, dashboard, files, terminal, memory, Matrix3D), and troubleshoot common issues.

SwitchUI is a React 19 + TanStack Start/Router application with a Hono BFF that runs on port 3002 and communicates with the Hermes API gateway (port 8642) and dashboard (port 9119).
