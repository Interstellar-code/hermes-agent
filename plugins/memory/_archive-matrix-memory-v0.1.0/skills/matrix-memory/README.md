# Matrix Memory bundled skill

This directory contains the bundled Hermes skill documentation for the Matrix Memory plugin.

## Contents

- `SKILL.md` — operator-facing instructions for configuring, invoking, and validating Matrix Memory workflows from Hermes.

## Intended use

Load this skill when the task is about Matrix Memory setup, recall usage, or operational guidance. Keep behavior claims grounded in the installed provider surface and local plugin documentation.

## Non-goals

- No provider implementation lives here.
- No tool schemas or runtime code are defined here.
- This folder should not be used to make storage-format or API promises that the provider code does not actually implement.
